# Доступный Маршрут 

Веб-приложение, которое строит не кратчайший, а **наиболее доступный маршрут** с учетом объектов доступной инфраструктуры и индивидуальных потребностей пользователя.

##  Цель проекта

Создание прототипа системы маршрутизации нового поколения, которая помогает людям с инвалидностью и ОВЗ безопасно и уверенно передвигаться по городу, учитывая:
-  Их конкретный тип ограничения мобильности
-  Наличие объектов доступной инфраструктуры на пути
-  Близость социальных объектов и учреждений
-  Качество улиц и наличие специального оборудования

##  Функциональность

###  Интеллектуальная маршрутизация
- **Персонализированное построение маршрутов** между двумя точками
- **Три режима маршрутизации** в зависимости от типа инвалидности:
  - ♿ **Для колясочников** - приоритет: пандусы, лифты, понижение бордюров, широкие двери
  - 👁️ **Для слабовидящих** - приоритет: тактильные плитки, звуковые светофоры, кнопки вызова
  - 🦯 **Для людей с тростью** - приоритет: поручни, ровные дороги, понижение бордюров

###  Навигация и информация
- **Геокодирование адресов** через Яндекс Maps API
- **Расчет расстояний** между точками с высокой точностью (формула Haversine)
- **Пошаговые инструкции** с описанием каждого участка маршрута
- **Время в пути** с учетом сниженной скорости передвижения (1.2 м/с)
- **Визуализация маршрута** на интерактивной карте

###  Социальная интеграция
- **Информация о социальных объектах**:
  - Медицинские учреждения
  - МФЦ и государственные учреждения
  - Места отдыха и развлечений
  - Культурные центры
- **Рекомендации близлежащих объектов** вдоль маршрута
- **Контактная информация** всех сервисов

###  Объекты доступной инфраструктуры
- **База данных объектов доступности**:
  - Пандусы (стационарные и откидные)
  - Лифты и подъемники для инвалидов
  - Тактильные плитки и направляющие
  - Звуковые светофоры
  - Кнопки вызова
  - Поручни и перила
  - Понижение бордюров
  - Расширенные входы и широкие двери

###  Голосовая поддержка
- **Синтез речи на русском языке** через Silero TTS
- **Голосовые подсказки** в реальном времени
- **Управление голосовым помощником** (включение/отключение)
- **Поддержка различных голосов и скоростей речи**

###  Пользовательский интерфейс
- **Адаптивный дизайн** для всех устройств (мобильные, планшеты, ПК)
- **Простая и интуитивная навигация**
- **Высокий контраст** и удобные размеры шрифтов
- **Поддержка скринридеров** и клавиатурной навигации

###  Система модерации
- **Модерация объектов доступности** - добавление новых объектов пользователями
- **Система одобрения** администратором
- **История изменений** и отслеживание активности
- **Рейтинг модераторов** и репутационная система

### 📊 Парсинг и обработка данных
- **Автоматический парсинг XML** с данными социальной поддержки
- **Обработка 500+ объектов** соцподдержки
- **Автоматическая категоризация** организаций
- **Географическая привязка** всех объектов через координаты



## 📊 Примеры использования

### Построение маршрута для колясочника
1. Выберите "♿ Для колясочников"
2. Введите "От" и "До"
3. Система найдет маршрут с пандусами и лифтами
4. Получите пошаговые инструкции и информацию о близких объектах

### Добавление нового объекта доступности
1. Перейдите на панель администратора
2. Заполните информацию об объекте
3. Загрузите фото (опционально)
4. Объект попадет на модерацию
5. После одобрения появится в базе
